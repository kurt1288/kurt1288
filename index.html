Road

A simple, small CSS framework

Download

Why Road?

Road is a simple, lightweight (only 2KB gzipped), and responsive CSS framework. It allows you to start working on the important details instead of spending time on the initial general setup.

This entire site is built with Road and only needed a few lines of custom CSS.

Want to contribute? Check out the Github repo!

Getting Started

First, download Road. Then just include both the Road and Normalize files in the header of your project.

What's included?

The download includes both unminified and minified versions of Road and Normalize as a reset.

Layout

Road's two layout options are a wrapping container and a flexible grid system.

Container

Adding a container wrapper can be done simply by adding a .container class to any element.

Three sizes of containers are available:

  • .container max width of 100rem
  • .container.wide max width of 140rem
  • .container.narrow max width of 60rem

Grid

Road's grid layout is based off the CSS flexbox. Each column will grow and shrink automatically as the device window changes. Unlike other CSS libraries that use arbitrary column widths, Road simply uses round percentages, like 10%, 40%, etc.

Typography

All typography is based off a starting font size of 10px. Headings are subsequently adjusted from this font size. Road uses three font weights: 200 for light, 300 for medium, and 400 for bold. In code elements, the font-family is Overpass Mono. The font families used are Overpass and Overpass Mono (for code elements). Both are created by Delve Fonts.

Buttons

Both button and input elements can be used and will be styled appropriately. Anchor elements can also be styled by adding the .button class.

Buttons may also be styled with the various colors by adding the corresponding class to the element.

Forms

Road hopefully makes your form-building experience a little easier. Styles are applied to all form elements, and when combined with the grid, laying out your form can be a breeze.

Tables

When building a table, make sure to use the thead and tbody markup appropriately. Tables can also be styled with the various colors by adding the desired class name to the table element.

Colors

Included in Road are various colors that can be applied to table, button, input elements, and anchor elements with the class .button. The color classes are as follows:

  • .black
  • .blue
  • .brown
  • .green
  • .orange
  • .purple
  • .red
  • .yellow
  • .yellowgreen

Utilities

The following utility classes are provided for convience:

Class Function
.clearfix Clear floated content
.f-left Float content left
.f-right Float content right
.f-none Set content float to none
.t-light Light (200) font weight
.t-normal Normal (400) font weight
.t-bold Bold (600) font weight
