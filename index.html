Why Road?

Looking for a lightweight CSS starting point, but don't need all the options available in a larger framework? Road gives you a easy place to start while leaving only a small footprint in your project.

Want to contribute? Check out the Github!

Grids

Road's grid layout is based off the CSS flexbox, with a maximum width of 100rem (1000px). Each column will grow and shrink automatically as the device window changes. Unlike other CSS libraries that use arbitrary column widths, Road simply uses round percentages, like 10%, 40%, etc.

Typography

Road uses rem for all font sizing. The base font size is 10px, so relative sizing can be done easily.

The font-family is Overpass, a font inspired by Highway Gothic (the font used on actual highway signs), and created by Delve Fonts. The font file is served by Google Fonts.

Road uses three font weights: 200 for light, 300 for medium, and 400 for bold.

In code elements, the font-family is Overpass Mono.

Buttons

Road offers buttons in many different styles, all emulating the colors of common traffic signs. Both button and input elements can be used and will be styled appropriately. Alternatively, anchor elements can be styled by adding the .button class.

Buttons may also be styled with the various colors by adding the corresponding class to the element.

Forms

Road hopefully makes your form-building experience a little eaiser. Styles are applied to all form elements, and when combined with the grid, laying out your form can be a breeze.

Colors

Included in Road are various colors that can be applied to table, button, input elements, and anchor elements with the class .button. The color classes are as follows:

  • .black
  • .blue
  • .brown
  • .green
  • .orange
  • .purple
  • .red
  • .yellow
  • .yellowgreen
